# Tela de Login com Bootstrap

**Alunos:** Mateus de Lima dos Santos, Natan Vitor de Andrade

## Descrição

Desenvolver uma tela de login estática utilizando o framework Bootstrap, aplicando conceitos de estruturação HTML e estilização com classes prontas.

Vocês deverão criar uma interface de login (apenas visual, sem funcionamento de backend) semelhante à que desenvolvemos em aula, porém agora utilizando os recursos do Bootstrap para estilização.

A tela deve ser responsiva e organizada, aproveitando os componentes e o sistema de grid do Bootstrap.

## Requisitos Funcionais

O sistema (tela) deve:

- Exibir um formulário de login contendo:
  - Campo de e-mail
  - Campo de senha
- Permitir que o usuário insira dados nos campos (interação básica do HTML)
- Exibir um botão de login
- Exibir um link auxiliar (ex: "Esqueci minha senha" ou "Criar conta")
- Apresentar todos os elementos organizados de forma clara e acessível
- Ser renderizado corretamente em navegador web

## Requisitos Não Funcionais

A aplicação deve atender aos seguintes critérios de qualidade:

### Usabilidade

- Interface limpa e intuitiva
- Campos identificados corretamente com labels
- Boa legibilidade (cores, contraste e espaçamento)

### Responsividade

- Layout adaptável para diferentes tamanhos de tela (desktop e mobile)
- Uso do sistema de grid do Bootstrap

### Padronização e Código

- Utilização adequada das classes do Bootstrap (`container`, `row`, `col`, `form-control`, `btn`, etc.)
- Código HTML bem estruturado e identado
- Uso correto de tags semânticas
